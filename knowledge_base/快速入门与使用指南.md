# MiniMax 快速入门与使用指南

## 接入前准备

接入 MiniMax 开放平台前，需完成以下三步准备：

1. **注册账户**：在 MiniMax 开放平台完成账户注册：https://platform.minimaxi.com/login
2. **创建 API Key**：进入密钥管理页面创建并保存可用的 API Key：https://platform.minimaxi.com/user-center/basic-information/interface-key
3. **账户充值**（按量付费时需要）：完成充值后方可按量调用：https://platform.minimaxi.com/user-center/payment/balance

详细文档：https://platform.minimaxi.com/docs/guides/quickstart-preparation

## API Key 类型与选择

MiniMax 提供两种 API Key，按用途选择：

| 类型 | 用途 | 支持模型 | 计费方式 |
|------|------|----------|----------|
| Token Plan Key | AI 编程工具 | 仅文本模型 | prompt 数量 |
| 普通 API Key | 所有 API 调用 | 全模态 | 按量计费（token/字符/次） |

**选择建议：**
- 主要在 Claude Code、Cursor 等编程工具中使用文本模型，优先选择 Token Plan Key
- 需要调用语音、视频、图像、音乐等多模态能力，使用普通 API Key

**获取方式：**
- Token Plan Key（以 `sk-cp` 开头）：在 https://platform.minimaxi.com/user-center/payment/token-plan 点击「重置并复制」获取，仅可调用文本模型
- 普通 API Key（以 `sk-api` 开头）：在 https://platform.minimaxi.com/user-center/basic-information/interface-key 创建，可调用文本、语音、视频、音乐、图像等全部模型接口

## 企业团队账号管理

企业团队推荐采用「主账号 + 子账号」方式管理：

- 子账号默认没有「支付」查看和管理权限，适合团队成员日常使用
- 子账号与主账号共享权益与速率限制
- 子账号管理入口：https://platform.minimaxi.com/user-center/basic-information/child-account

## MiniMax MCP 工具

MiniMax MCP（模型上下文协议）工具允许在 AI 编程工具中直接调用 MiniMax 的 API 能力。

**官方实现：**
- Python 版本：https://github.com/MiniMax-AI/MiniMax-MCP
- JavaScript 版本：https://github.com/MiniMax-AI/MiniMax-MCP-JS

详细文档：https://platform.minimaxi.com/docs/guides/mcp-guide

## MiniMax MCP 内置工具列表

MiniMax MCP 内置以下常用工具：

| 工具 | 功能 |
|------|------|
| text_to_audio | 文本合成语音 |
| list_voices | 查询可用音色 |
| voice_clone | 音频文件克隆音色 |
| voice_design | 提示词生成音色 |
| play_audio | 播放音频文件 |
| music_generation | 提示词和歌词生成音乐 |
| generate_video | 文本或图片生成视频 |
| image_to_video | 首帧图像生成视频（JS 版本） |
| query_video_generation | 查询视频生成任务状态 |
| text_to_image | 提示词生成图片 |

## Token Plan MCP 专属能力

Token Plan MCP 在标准 MCP 工具之外，额外提供两项专属工具：

1. **网络搜索**：在编码过程中快速检索信息
2. **图片理解**：识别并理解图片内容

详细文档：https://platform.minimaxi.com/docs/guides/token-plan-mcp-guide

## MiniMax 开放平台 API 能力概览

MiniMax 开放平台提供以下 API 能力：

- **文本**：Anthropic 兼容 / OpenAI 兼容 / 文本对话 / Prompt 缓存
- **语音**：同步合成（HTTP/WebSocket）/ 异步合成 / 音色复刻 / 音色设计 / 音色管理
- **视频**：文生视频 / 图生视频 / 首尾帧 / 主体参考 / 视频 Agent
- **图像**：文生图 / 图生图
- **音乐**：音乐生成 / 歌词生成
- **文件管理**：上传 / 检索 / 列出 / 下载 / 删除

API 完整文档：https://platform.minimaxi.com/docs/api-reference/api-overview
