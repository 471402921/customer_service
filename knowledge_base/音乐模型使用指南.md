# MiniMax 音乐模型使用指南

## MiniMax 音乐模型有哪些，适合哪些创作场景？

MiniMax 开放平台提供以下音乐生成模型：

| 模型名称 | 介绍 |
|---------|------|
| music-2.5+ | 最新音乐生成模型，支持纯音乐，突破风格边界 |
| music-2.5 | 最新音乐生成模型，全维度突破，指挥细节，定义真实 |
| music-2.0 | 多变音色，丰富乐器表现 |

## Music 2.5 相比前代模型有哪些核心提升？

Music 2.5 在以下四个维度相比 Music 2.0 全面升级：

- **编曲混音**：扩展高采样率音色库（含管弦、民乐等），优化声场算法
- **人声表现**：优化 AI 合成音痕迹，引入拟人化音色模拟，Flow 表现力增强
- **结构精度**：支持 Intro/Bridge/Interlude/Build-up/Hook 等 14 种以上结构标签
- **声音设计**：自动识别并还原目标风格的关键物理特性

## 音乐生成接口如何调用？关键参数有哪些？

- 接口文档：https://platform.minimaxi.com/docs/api-reference/music-generation
- 使用指南：https://platform.minimaxi.com/docs/guides/music-generation
- 请求端点：`POST https://api.minimaxi.com/v1/music_generation`

关键请求参数：

| 参数 | 必填 | 说明 |
|-----|:---:|------|
| model | 是 | 模型名称，可选 music-2.5+ / music-2.5 / music-2.0 |
| prompt | 是 | 音乐风格、情绪、场景描述（英文效果更佳） |
| lyrics | 否 | 歌词内容，可含结构标签如 [Intro][Verse][Chorus] |
| audio_setting.sample_rate | 否 | 采样率，默认 44100 |
| audio_setting.bitrate | 否 | 比特率，默认 256000 |
| audio_setting.format | 否 | 输出格式：mp3（默认）或 wav |
| output_format | 否 | 返回方式：url 或 base64 |

## 音乐生成接口调用示例

```python
import requests, os

url = "https://api.minimaxi.com/v1/music_generation"
headers = {
    "Content-Type": "application/json",
    "Authorization": f"Bearer {os.environ.get('MINIMAX_API_KEY')}"
}

payload = {
    "model": "music-2.5",
    "prompt": "Mandopop, Festive, Upbeat, Celebration",
    "lyrics": "[Intro]\n嘿！新年到！\n[Verse]\n...",
    "audio_setting": {
        "sample_rate": 44100,
        "bitrate": 256000,
        "format": "mp3"
    },
    "output_format": "url"
}

response = requests.post(url, headers=headers, json=payload)
```

## 歌词生成接口如何使用？

MiniMax 提供独立的歌词生成接口，可自动生成带结构标签的完整歌词。

- 接口文档：https://platform.minimaxi.com/docs/api-reference/lyrics-generation
- 请求端点：`POST https://api.minimaxi.com/v1/lyrics_generation`

关键请求参数：

| 参数 | 必填 | 说明 |
|-----|:---:|------|
| mode | 是 | write_full_song（完整歌曲创作）或 edit（编辑/续写） |
| prompt | 是 | 歌词创作提示，描述主题、风格、情感等 |

## 歌词生成接口调用示例

```python
import requests, os

url = "https://api.minimaxi.com/v1/lyrics_generation"
headers = {
    "Content-Type": "application/json",
    "Authorization": f"Bearer {os.environ.get('MINIMAX_API_KEY')}"
}

payload = {
    "mode": "write_full_song",
    "prompt": "一首欢乐的新年歌曲"
}

response = requests.post(url, headers=headers, json=payload)
```

## 音乐模型如何计费？

MiniMax 音乐模型按生成次数计费：

| 模型 | 单价 |
|-----|------|
| music-2.5+ | 1.0 元/首 |
| music-2.5 | 1.0 元/首 |
| music-2.0 | 0.25 元/首 |
| 歌词生成 | 0.05 元/次 |

每次调用音乐生成接口生成一首歌曲，计费一次；歌词生成接口每次调用单独计费。
