# 文件管理使用指南

MiniMax 开放平台提供文件管理接口，用于上传、检索、列出、下载和删除文件。

- Base URL：`https://api.minimaxi.com`
- 认证方式：Bearer Token（API Key）
- 接口文档总览：https://platform.minimaxi.com/docs/api-reference/file-management-upload

---

## 文件上传

文件上传接口用于将本地文件上传至 MiniMax 平台，供后续 API 调用使用。

- 接口文档：https://platform.minimaxi.com/docs/api-reference/file-management-upload
- 端点：`POST /v1/files/upload`
- Content-Type：`multipart/form-data`

**请求参数：**

| 参数 | 必填 | 说明 |
|------|------|------|
| purpose | 是 | 文件用途：voice_clone / prompt_audio / t2a_async_input |
| file | 是 | 文件路径 |

**purpose 与支持格式对照：**

| purpose | 支持格式 |
|---------|---------|
| voice_clone | mp3, m4a, wav |
| prompt_audio | mp3, m4a, wav |
| t2a_async_input | text, zip |

- `voice_clone`：上传用于声音克隆的音频文件，支持 mp3、m4a、wav 格式
- `prompt_audio`：上传用于提示音频的文件，支持 mp3、m4a、wav 格式
- `t2a_async_input`：上传用于异步文本转语音的输入文件，支持 text、zip 格式

---

## 文件列表查询

文件列表接口用于列出指定用途分类下已上传的全部文件。

- 接口文档：https://platform.minimaxi.com/docs/api-reference/file-management-list
- 端点：`GET /v1/files/list?purpose={purpose}`

通过 `purpose` 参数过滤文件类型，返回该分类下所有文件的列表信息。

---

## 文件信息检索

文件检索接口用于通过 file_id 查询单个文件的详细信息和下载地址。

- 接口文档：https://platform.minimaxi.com/docs/api-reference/file-management-retrieve
- 端点：`GET /v1/files/retrieve?file_id={file_id}`

**支持检索的 file_id 来源：**

- 视频生成任务状态查询返回的 file_id
- 异步语音合成任务状态查询返回的 file_id

**返回信息包含：**

- `download_url`：文件下载 URL（视频文件的下载 URL 有效期为 1 小时，请及时下载）

---

## 文件内容直接下载

文件内容下载接口用于直接拉取文件的二进制内容。

- 接口文档：https://platform.minimaxi.com/docs/api-reference/file-management-retrieve-content
- 端点：`GET /v1/files/retrieve_content?file_id={file_id}`

该接口返回文件的原始二进制内容，适合直接保存文件使用。

---

## 文件删除

文件删除接口用于删除平台上已上传的文件。

- 接口文档：https://platform.minimaxi.com/docs/api-reference/file-management-delete
- 端点：`POST /v1/files/delete`

**请求参数：**

| 参数 | 必填 | 说明 |
|------|------|------|
| file_id | 是 | 文件唯一标识符 |
| purpose | 是 | 文件用途：voice_clone / prompt_audio / t2a_async / t2a_async_input / video_generation |

---

## 常见错误码说明

完整错误码列表：https://platform.minimaxi.com/docs/api-reference/errorcode

| 状态码 | 说明 |
|--------|------|
| 0 | 请求成功 |
| 1000 | 未知错误 |
| 1001 | 超时 |
| 1002 | 触发 RPM 限流（每分钟请求次数超限） |
| 1004 | 鉴权失败（API Key 无效或未携带） |
| 1008 | 余额不足 |
| 1013 | 服务内部错误 |
| 1026 | 输入内容涉及敏感内容 |
| 1027 | 输出内容涉及敏感内容 |
| 1039 | 触发 TPM 限流（每分钟 Token 数超限） |
| 2013 | 传入参数异常（请检查请求参数格式是否正确） |
| 2049 | 无效的 API Key |
