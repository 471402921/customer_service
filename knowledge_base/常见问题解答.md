# MiniMax 开放平台常见问题解答

本文档汇总 MiniMax 开放平台(国内域名 `minimaxi.com`,多个 i)的高频常见问题(FAQ),覆盖账户与计费、模型与功能、Token Plan 套餐、错误码与速率限制、官方联系方式等主题。每条 Q&A 均独立完整,可单独检索使用。

## 账户与计费

### Q. MiniMax 开放平台支持哪些充值方式?到账时间多久?

MiniMax 开放平台支持两种充值方式:在线充值与对公汇款。

- **在线充值**:前往「账户管理 > 余额」页面,使用微信或支付宝完成在线充值。充值入口:https://platform.minimaxi.com/user-center/payment/balance
- **对公汇款**:仅企业用户支持。汇款账户名称必须与平台实名认证主体一致;到账后通常 10 分钟至 1 小时入账。

详细文档:https://platform.minimaxi.com/docs/faq/about-account

### Q. 如何获取 MiniMax 开放平台的 GroupId?

登录 MiniMax 开放平台,前往「账户管理 > 账户信息」页面即可查看并复制 GroupId。页面入口:https://platform.minimaxi.com/user-center/basic-information

GroupId 是调用 MiniMax API 时必填的账户标识,用于区分不同账户主体。

### Q. MiniMax 开放平台发票如何申请?有哪些限制条件?

MiniMax 开放平台发票申请规则与限制如下:

- 发票抬头必须与实名认证主体一致。
- 个人账号不能直接开具企业抬头发票;如需企业发票,需先完成企业实名认证或补充对应企业信息。
- 可开票金额 = 已消耗金额 - 已开票金额(资源包未消耗部分及账户余额不可开票)。
- 申请入口:https://biaodan.info/web/formview/6503f8ba75a03c4a8388a91a

### Q. MiniMax 开放平台是否支持余额预警提醒?如何设置?

支持余额预警提醒。设置方法:

1. 登录 MiniMax 开放平台,进入「账户管理 > 余额」页面:https://platform.minimaxi.com/user-center/payment/balance
2. 设置预警金额阈值。
3. 当账户余额低于阈值时,系统会通过邮件、短信、站内信三种渠道发送提醒。

### Q. MiniMax 开放平台资源包有效期和过期规则是什么?

MiniMax 开放平台资源包规则:

- 有效期可在「账户管理 > API 资源包」页面查看,入口:https://platform.minimaxi.com/user-center/payment/subscription
- **资源包余量不继承**:同一类型资源包过期后,未消耗的余量不会延续到新购资源包。
- **过期自动清零**:有效期截止后,剩余额度自动清零,不可恢复。

## 模型与功能可用性

### Q. MiniMax 开放平台支持 Embedding 模型接入吗?收费规则是什么?

MiniMax 开放平台目前**暂时不提供 Embedding 模型服务**。如有相关需求,请关注后续平台公告。

### Q. MiniMax 开放平台有语音识别(ASR)功能吗?ASR 服务怎么配置?

MiniMax 开放平台**目前暂不提供语音识别(ASR)服务**。如需语音识别能力,可自行安装开源免费的 Whisper ASR Skill 或其他第三方 ASR 方案。

### Q. Token Plan Key 和普通 API Key 有什么区别?可以混用吗?

Token Plan Key 与普通 API Key 是两种独立的鉴权凭证,**两种 Key 不能混用**,需根据使用场景选择。

- **Token Plan Key**:通过订阅 Token Plan 套餐后创建,**仅适用于文本模型(编程工具场景)**,按 prompt 数量计费。
- **普通 API Key**:在 API 密钥管理页面创建,**适用于所有模态**(文本、语音、图像、视频、音乐),按量计费(token 用量)。

详细文档:https://platform.minimaxi.com/docs/faq/about-apis

## 错误码与速率限制

### Q. MiniMax API 常见错误码怎么排查?

MiniMax API 常见错误码及处理建议如下:

- **0**:请求成功。
- **1002**(触发限流):降低请求频率,或申请提升速率限制。
- **1004**(鉴权失败):检查 API Key 是否正确、是否过期、是否与 GroupId 匹配。
- **1008**(余额不足):为账户充值。
- **1013**(服务内部错误):稍后重试,如持续出现请联系官方客服。
- **1026**(输入内容涉及敏感内容):调整输入内容,去除敏感词。
- **1027**(输出内容涉及敏感内容):调整输入提示,引导生成符合规范的输出。
- **1039**(触发 TPM 限流):降低 token 消耗速率,或申请提升 TPM 配额。
- **2013**(传入参数异常):检查请求参数格式与必填项。
- **2049**(无效的 API Key):检查 API Key 是否正确、是否被删除。

完整错误码列表:https://platform.minimaxi.com/docs/api-reference/errorcode

### Q. 如何申请提高 MiniMax API 速率限制?

申请提高 MiniMax API 速率限制的方式:

- 联系 MiniMax 官方客服(微信扫描官网底部二维码)。
- 发送邮件至 api@minimaxi.com 提交申请。

处理周期一般为 **3-5 个工作日**,审核通过后会主动通知用户。

## Token Plan 套餐

### Q. MiniMax Token Plan 支持哪些模型?

Token Plan 各套餐支持的模型如下:

- **文本模型**:MiniMax-M2.7(标准版),MiniMax-M2.7-highspeed(极速版)。
- **语音模型**:Speech 2.8(Plus 及以上套餐)。
- **图像模型**:image-01(Plus 及以上套餐)。
- **视频模型**:Hailuo-2.3-Fast 768P 6s、Hailuo-2.3 768P 6s(Max 及以上套餐)。
- **音乐模型**:Music-2.6(所有套餐均可使用,限时免费推广,100 首/天)。

各模型额度因套餐档位不同而异,详见 Token Plan 介绍页:https://platform.minimaxi.com/docs/token-plan/intro

### Q. Token Plan 极速版和标准版有什么区别?

Token Plan 极速版与标准版的主要区别:

- **标准版**:支持 MiniMax-M2.7 文本模型。
- **极速版**:在标准版基础上,**额外支持 MiniMax-M2.7-highspeed**,推理速度更快,生成效果与 MiniMax-M2.7 一致。

如对响应延迟敏感,推荐选择极速版。

### Q. Token Plan 可以随时升级套餐吗?

可以。Token Plan **支持随时升级套餐**,包括从标准版升级到极速版,以及套餐档位之间的升级。

升级规则:**补差价后立即生效**,无需等待当前周期结束。

### Q. Token Plan 中"每 5 小时重置"是怎么计算的?

"每 5 小时重置"属于**动态速率限制机制**,而非固定时间窗口。

计算规则:**5 小时前消耗的额度会自动释放回可用额度池**。即任意时刻可用的额度 = 套餐总额度 - 过去 5 小时内已消耗的额度。

举例:某用户在 10:00 消耗了 50 个 prompts,这 50 个额度会在 15:00 自动释放回可用池。

### Q. Token Plan 达到限额后可以怎么处理?

Token Plan 达到限额后,有以下三种处理方式:

1. **升级套餐**:补差价升级到更高档位,立即获得更高额度。
2. **切换到按量付费**:更换为普通 API Key,使用按量计费模式继续调用。
3. **等待 5 小时窗口滚动重置**:5 小时前消耗的额度会自动释放,无需操作。

### Q. 如何查看 Token Plan 自动续费状态?如何取消自动续费?

查看与取消 Token Plan 自动续费有两种方式:

**方法一(在支付宝侧操作):**

1. 打开支付宝 App,进入「我的」>「设置」。
2. 选择「支付设置」>「免密支付/自动扣款」。
3. 找到 MiniMax 相关订阅项,可查看当前状态或关闭自动扣款。

**方法二(在 MiniMax 开放平台查看状态):**

1. 登录 MiniMax 开放平台,进入「用户中心」>「Token Plan」页面。
2. 查看当前订阅状态。
3. 如需取消自动续费,**请在支付宝侧关闭自动扣款**(平台侧仅展示状态,不直接控制扣款)。
