# MiniMax 常见问题解答

## 账户与充值

### 支持哪些充值方式，到账时间多久？

MiniMax 开放平台支持两种充值方式：

- **在线充值**：前往「账户管理 > 余额」使用微信充值，充值后即时到账：https://platform.minimaxi.com/user-center/payment/balance
- **对公汇款**：仅企业用户支持；汇款账户名称需与平台实名认证主体一致；到账后通常 10 分钟~1 小时内入账

详细文档：https://platform.minimaxi.com/docs/faq/about-account

### 如何获取 GroupId？

前往「账户管理 > 账户信息」页面复制 GroupId：https://platform.minimaxi.com/user-center/basic-information

### 发票如何申请，有哪些限制条件？

- 发票抬头需与实名认证主体一致
- 个人账号不能直接开企业抬头发票（需先完成企业认证）
- 可开票金额 = 已消耗金额 - 已开票金额
- 发票申请入口：https://biaodan.info/web/formview/6503f8ba75a03c4a8388a91a

### 是否支持余额预警提醒？

支持余额预警提醒。可在「账户管理 > 余额」设置预警金额；余额低于阈值时会通过邮件、短信、站内信发送提醒：https://platform.minimaxi.com/user-center/payment/balance

### 资源包有效期和过期规则是什么？

- 有效期可在「账户管理 > API 资源包」查看：https://platform.minimaxi.com/user-center/payment/subscription
- 资源包余量不继承，过期自动清零

## API Key 相关

### Token Plan Key 和普通 API Key 有什么区别？

两种 Key 不能混用，按实际用途选择：

| 类型 | 获取方式 | 适用范围 | 计费 |
|------|----------|----------|------|
| Token Plan Key | 创建 Token Plan Key | 仅文本模型（编程工具） | prompt 数量 |
| 普通 API Key | 创建新的 API Key | 所有模态 | 按量计费 |

详细文档：https://platform.minimaxi.com/docs/faq/about-apis

### 如何申请提高 API 速率限制？

可联系官方客服，或发送邮件到 api@minimaxi.com 申请提速；处理周期一般为 3-5 个工作日。

## 常见错误码排查

| 错误码 | 含义 | 解决方案 |
|--------|------|----------|
| 0 | 请求成功 | — |
| 1002 | 触发限流 | 降低请求频率或提升速率限制 |
| 1004 | 鉴权失败 | 检查 API Key 是否正确 |
| 1008 | 余额不足 | 充值账户余额 |
| 1013 | 服务内部错误 | 稍后重试 |
| 1026 | 输入内容涉及敏感内容 | 调整输入内容 |
| 1027 | 输出内容涉及敏感内容 | 调整输入内容 |
| 1039 | 触发 TPM 限流 | 降低 token 消耗速率 |
| 2013 | 传入参数异常 | 检查请求参数 |
| 2049 | 无效的 API Key | 检查 API Key |

完整错误码列表：https://platform.minimaxi.com/docs/api-reference/errorcode

## Token Plan 订阅相关

### Token Plan 支持哪些模型？

Token Plan 支持以下模型：
- `MiniMax-M2.5`
- `MiniMax-M2.1`
- `MiniMax-M2`

极速版额外支持 `MiniMax-M2.5-highspeed`。

详细文档：https://platform.minimaxi.com/docs/token-plan/faq

### 极速版和标准版有什么区别？

极速版额外支持 `MiniMax-M2.5-highspeed`，推理速度更快，效果与 M2.5 一致。

### Token Plan 可以随时升级套餐吗？

支持随时升级套餐（含标准版升极速版），补差价后立即生效。

### "每 5 小时重置"是怎么计算的？

Token Plan 使用动态速率限制机制：5 小时前消耗的额度会自动释放，属于滚动窗口重置，不是固定时间点重置。

### 达到限额后可以怎么处理？

达到 Token Plan 限额后有三种处理方式：

1. 升级套餐（提高总额度）
2. 切换到按量付费（更换为普通 API Key）
3. 等待 5 小时窗口滚动重置

### Token Plan 支持退款吗？

不支持退款。
