# 开平客服需知信息

本文档覆盖 MiniMax 开放平台账户权益相关常见问题，包括开票、账户与企业认证、接口报错排查、代金券、退款、Token Plan 升级杂项。国内开放平台域名为 `minimaxi.com`（多个 i），海外 MiniMax International 域名为 `minimax.io`，二者不可混用。

## 开票

### Q. MiniMax 开放平台怎么自助开票？开票入口在哪里？

MiniMax 开放平台全平台开票均支持自助开具，无需人工处理。不同充值类型的开票入口如下：

- Token 套餐开票入口：https://platform.minimaxi.com/user-center/payment/token-plan
- 余额开票入口：https://platform.minimaxi.com/user-center/payment/invoice

### Q. MiniMax 开放平台开票时需要注意什么关键信息？group id 在哪里查？

MiniMax 开放平台开票时，**group id 必须填写准确**，否则会影响开票进度。

group id 可在用户中心基本信息页面核实校验：https://platform.minimaxi.com/user-center/basic-information

### Q. MiniMax 开放平台发票抬头或发票类型有误，怎么重开发票？

MiniMax 开放平台发票抬头或类型有误时，需要先对原发票进行红冲，再重新开具新发票。

如果是**普通发票换开增值税专用发票**，还需补充以下企业资质信息：

- 开户银行
- 银行账号
- 公司地址
- 公司电话
- 资质证明（以下三选一上传）：历史开具专票图片、税局系统查询为一般纳税人的图片、一般增值税纳税人证书
- 上传委托证明：增值税专用发票申请委托及承诺书

### Q. MiniMax 开放平台提交开票申请后长时间没收到发票怎么办？

MiniMax 开放平台未收到发票通常是以下两种原因，可自行核对修正：

- 开票申请页面操作错误
- 邮箱等开票信息填写有误

### Q. MiniMax 新支付的订单多久可以开具并收到发票？

MiniMax 平台已支持全自助开票，用户可自行申请开具发票，发票一般在 1~3 个工作日内可收到。

## 账户与企业认证

### Q. MiniMax 开放平台入口有哪些？国内和海外平台分别怎么进？

MiniMax 开放平台入口区分国内和海外两个独立站点：

- 国内开放平台：https://platform.minimaxi.com
- 海外开放平台：https://www.minimax.io/platform

**海外开放平台仅支持谷歌邮箱登录**，建议使用便于登录的谷歌邮箱进行注册。

### Q. MiniMax 开放平台账户如何注册？

MiniMax 开放平台目前支持**手机号**完成账户注册。注册完成后可后续绑定邮箱等方式，便于账号认证与找回。

海外平台 MiniMax International 注册请使用谷歌邮箱登录。

### Q. MiniMax 开放平台企业认证怎么操作？认证入口在哪里？

MiniMax 开放平台企业认证入口：https://platform.minimaxi.com/user-center/basic-information

企业认证流程为：校验认证信息 → 填写绑定账户信息 → 填写鉴权金额 → 认证完成。

企业认证注意事项：

- 认证银行信息与打款银行信息需保持一致
- 信息变更后不支持换绑
- 若报错「该待绑定账户已绑定在其他会员子账户上 (1016)」，说明该银行卡已被绑定，建议更换银行卡重新绑定

### Q. MiniMax 企业认证后打款金额被退回，回款凭证显示"交易对手不在白名单内"是什么问题？

该报错原因是当前交易行为中的打款银行和之前 MiniMax 企业认证的银行不是同一个。需要使用**认证时所用的银行**进行对公打款操作，才能完成鉴权。

### Q. MiniMax 开放平台账号使用有什么限制？可以换绑手机号吗？

MiniMax 开放平台目前实行**一号一户**策略，不建议进行手机号换绑操作。

### Q. MiniMax 开放平台账户如何注销？注销后有什么影响？

MiniMax 开放平台账户注销需要将以下信息发送至邮箱 `nashi@minimaxi.com`，工作人员将协助操作注销：

- 姓名
- group id
- 手机号

注意：MiniMax 开放平台账户一旦注销，将**无法再次注册使用**，请谨慎操作。

## 接口报错排查（提交工单需提供的信息）

### Q. MiniMax 视频接口报错，需要提供什么信息排查？

MiniMax 视频接口出现报错，**必须提供 task id**，才能精准定位问题。请在提交工单或联系客服时附上对应的 task id。

### Q. MiniMax 语音接口、音乐接口报错，需要提供什么信息排查？

MiniMax 语音接口和音乐接口报错，**必须提供 Trace ID**，才能定位问题。请在提交工单或联系客服时附上对应的 Trace ID。

### Q. MiniMax 接口的 Trace ID 怎么获取？

MiniMax 接口的 Trace ID 直接在代码中打印响应头即可获取，参考代码：

```plaintext
print("Trace-ID:", response.headers['Trace-ID'])
```

获取到的 Trace ID 用于 MiniMax 语音、音乐等接口的报错排查。

### Q. MiniMax M 系列接口报错，需要提供什么信息排查？

MiniMax M 系列接口报错，**最好提供 request id**（作用与 Trace ID 类似），用于问题定位。

## 代金券

### Q. MiniMax 试用代金券是什么？怎么获得？能用来干什么？

MiniMax 试用代金券是在 MiniMax 开放平台完成**个人认证或企业认证**后获得的代金券。

试用代金券使用范围：

- 可用于：MiniMax 开放平台所有模型调用消耗，优先抵扣
- 不可用于：购买资源包（含 Coding Plan）

### Q. MiniMax 激励代金券是什么？怎么获得？能用来干什么？

MiniMax 激励代金券通过「Coding Plan 共建邀请计划」获得，入口：https://platform.minimaxi.com/user-center/basic-information

激励代金券使用范围：

- 可用于：MiniMax 开放平台所有模型调用消耗，优先抵扣
- 也可用于：购买资源包（含 Coding Plan）

### Q. MiniMax 平台代金券有哪几种？各自有效期是多久？

MiniMax 平台代金券共有 5 种来源及对应有效期：

- **注册认证代金券**：面额 15 元，从**触发使用**起有效期 3 天
- **邀请返利代金券**：从**获得当日**起有效期 90 天
- **直播活动代金券**：有效期 3–7 天不等
- **模型内测专属代金券**：内部下发给优质开发者，有效期 3–7 天
- **补偿代金券**：因平台计费问题补偿用户，有效期一般为 90 天

## 退款

### Q. MiniMax 如何申请退款？退款流程是怎样的？

MiniMax 退款需通过官方申请通道提交，分为**套餐退款**和 **API 余额退款**两种情况。提交后进入人工审核，是否可退、可退金额以官方最终审核结果为准。

不同退款类型对应不同的申请表，详见下方问题。

### Q. MiniMax 购买了套餐要退款，该怎么申请？

MiniMax 套餐退款适用于 Max 极速版、各类 Token Plan、Token Plan 订阅等套餐，退款规则如下：

- 退款金额**按天核算**，以套餐生效日至提交退款申请日为已使用周期，扣除已使用天数费用后退还剩余金额
- 退款以**实际支付金额**为基数，购买时使用的**代金券不予返还**
- 提交退款申请后 **1-2 个工作日内**将回收相关权益，若期间仍有权益任务消耗，将默认取消本次退款申请
- 退款将在 **7-15 个工作日**内到账；合并支付（使用部分代金券/余额）退款会慢 3-5 天

请填写《MiniMax Token Plan 用户权益变更表》：https://vrfi1sk8a0.feishu.cn/share/base/form/shrcnj5q5gfC4KkUxLcQXD9IDsn

### Q. MiniMax 开放平台 API 账户余额如何退款？

MiniMax API 账户余额退款适用于 API 账户充值余额的退款场景。

请填写《MiniMax 开放平台余额退款申请表》：https://vrfi1sk8a0.feishu.cn/share/base/form/shrcnVNudsGqI6XQvF1dy2Iv4p7

### Q. MiniMax 填写退款申请表需要附带哪些信息？

填写 MiniMax 退款申请表时，建议一并提交以下信息以便审核：

- MiniMax 账号
- 订单号或支付截图
- 退款原因说明

## Token Plan 升级与订阅杂项

### Q. MiniMax Token Plan 想要降级订阅套餐，该如何操作？

MiniMax Token Plan 套餐权益**暂不支持直接降级**。

若需更换更低档位套餐，操作步骤为：先在**支付宝订阅管理**中关闭自动续订，待当前套餐周期结束后，再重新选择合适额度的套餐即可。

### Q. MiniMax Token Plan 升级后会员时长为何没有延长？

MiniMax Token Plan 套餐平级续订或升级时，原套餐剩余权益会**折算抵扣新套餐部分费用**，新套餐立即生效，因此并非按原价续订。同时原套餐周期将终止，升级后将重新计算新的订阅自然周期。

这是为什么 MiniMax Token Plan 升级后会员时长不会简单地"叠加"在原有时长上。

### Q. MiniMax Token Plan 老用户原本没有使用限制，升级后会受周额度限制吗？

MiniMax Token Plan 老用户升级后是否受周额度限制，按购买时间分两类：

- **2026 年 3 月 22 日 23:59:59 前购买的用户**：不受周额度限制，平级续订或升级后仍不受该限制影响
- **2026 年 3 月 23 日 00:00:00 起购买的用户**：需遵守周额度限制规则

周额度限制仅针对 M2.x 系列文本生成模型。

结论：MiniMax Token Plan 原来没有周限制的老用户，在升级和平级续订后都**不会**受周限制影响。
