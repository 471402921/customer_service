# 语音模型使用指南

## MiniMax 有哪些语音模型，分别适合什么需求？

MiniMax 提供以下语音合成模型：

- **speech-02-hd**：高清音质，适合对音质要求高的场景，如有声书、播客、配音等内容创作
- **speech-02-turbo**：快速响应，适合实时对话、低延迟场景，如语音助手、客服机器人
- **speech-01-hd**：上一代高清模型，稳定可用
- **speech-01-turbo**：上一代快速模型，稳定可用
- **speech-01-240228**：早期版本，兼容存量业务

推荐优先使用 speech-02 系列。

## 同步语音合成

### 同步语音合成适合哪些场景，如何调用？

同步语音合成（Text-to-Audio，简称 T2A）适合**实时或低延迟场景**，支持文本到语音的即时生成。

**核心特性：**
- 单次最长处理 **10,000 字符**
- 支持 HTTP 与 WebSocket 两种协议
- WebSocket 支持流式返回（边生成边播放）
- 支持音量、语调、语速、混音等参数控制
- 支持比特率、采样率等音频参数设置
- 返回音频时长、音频大小等元信息

**API 端点：**
- HTTP：`POST https://api.minimaxi.com/v1/t2a_v2`
- WebSocket：`wss://api.minimaxi.com/ws/v1/t2a_v2`

**接口文档：**
- HTTP 接口：https://platform.minimaxi.com/docs/api-reference/speech-t2a-http
- WebSocket 接口：https://platform.minimaxi.com/docs/api-reference/speech-t2a-websocket
- 使用指南：https://platform.minimaxi.com/docs/guides/speech-t2a-websocket

### 同步语音合成的关键参数有哪些？

同步语音合成 `/v1/t2a_v2` 接口的关键参数说明：

| 参数 | 类型 | 说明 |
|------|------|------|
| model | string | 模型名称，如 `speech-02-hd`、`speech-02-turbo` |
| text | string | 待合成的文本内容，最长 10,000 字符 |
| voice_id | string | 音色 ID，支持系统音色或自定义复刻音色 |
| speed | float | 语速，范围 0.5～2.0，默认 1.0 |
| vol | float | 音量，范围 1～10，默认 1.0 |
| pitch | int | 语调，范围 -12～12，默认 0 |
| audio_sample_rate | int | 采样率，支持 16000、22050、24000、32000、44100 |
| bitrate | int | 比特率，支持 32000、64000、128000 |
| format | string | 输出格式，支持 mp3、wav、flac、pcm |
| stream | bool | 是否流式返回（WebSocket 场景下使用） |

## 异步语音合成

### 异步语音合成和同步方式有什么区别，怎么用？

异步语音合成更适合**长文本任务**，不要求即时返回。

**核心特性：**
- 单个文件长度限制 **< 10 万字符**
- 支持传字符串或上传文本文件
- 支持返回句级时间戳（可生成字幕文件）
- 结果通过下载 URL 获取，URL 有效期 **9 小时**

**接口文档：**
- 创建任务：https://platform.minimaxi.com/docs/api-reference/speech-t2a-async-create
- 查询任务状态：https://platform.minimaxi.com/docs/api-reference/speech-t2a-async-query
- 使用指南：https://platform.minimaxi.com/docs/guides/speech-t2a-async

### 异步语音合成的标准调用流程是什么？

1. （可选）调用文件上传 API，将文本文件上传至平台，获取 `file_id`
2. 调用**创建语音生成任务** API，传入文本或 `file_id`，获取 `task_id`
3. 调用**查询任务状态** API，使用 `task_id` 轮询，直到任务状态变为完成
4. 使用返回的 `file_id` 调用**文件下载** API，获取合成音频

## 音色复刻

### 音色快速复刻怎么做？有哪些限制？

音色快速复刻允许上传参考音频，克隆指定说话人的音色。

**操作步骤：**
1. 上传待克隆的参考音频
   - 支持格式：mp3、m4a、wav
   - 时长：10 秒 ~ 5 分钟
   - 大小：≤ 20MB
2. （可选）上传示例音频，以增强复刻相似度和稳定性
   - 时长：< 8 秒
3. 调用音色复刻接口，基于 `file_id` 和自定义 `voice_id` 完成克隆
4. 在语音合成请求中使用该 `voice_id`

**重要限制：** 如果复刻的音色 **7 天内未正式调用**，系统会自动删除该音色。

**接口文档：**
- 复刻接口：https://platform.minimaxi.com/docs/api-reference/voice-cloning-clone
- 使用指南：https://platform.minimaxi.com/docs/guides/speech-voice-clone

### 音色快速复刻如何计费？

- 音色复刻费用：**9.9 元/音色**
- 计费时间：在**首次使用该复刻音色进行语音合成**时收取，非复刻时收取

## 音色设计

### 音色设计是什么，如何使用和计费？

音色设计可通过**文本 Prompt** 描述音色特征，自动生成新音色（返回 `voice_id`），无需上传参考音频。

**计费规则：**
- 音色设计费用：**9.9 元/音色**，在**首次使用该设计音色进行语音合成**时收取
- 试听语音合成按 **2 元/万字符** 收费

**接口文档：**
- 音色设计接口：https://platform.minimaxi.com/docs/api-reference/voice-design-design
